# User Story: 検索APIエンドポイント

> **Epic**: APIプロキシサーバー移行
> **作成日**: 2026-02-11

---

## User Story

ユーザーとして、アプリから動画やチャンネルを検索したい。なぜなら、APIキーを安全にサーバー側で管理しつつ、YouTube / Twitchの横断検索ができるようにしたいから。

---

## ゴール

YouTube API / Twitch APIへのプロキシとしてマルチサービス動画検索とチャンネル検索のエンドポイントを実装する

---

## 依存

- US-1: 共通基盤 & サーバー骨格構築（US-2と並行可能）

---

## スコープ

- マルチサービス動画検索エンドポイント（`GET /api/search/videos`）
- チャンネル検索エンドポイント（`GET /api/search/channels`）
- YouTube Search APIへのプロキシ呼び出し実装
- Twitch Search APIへのプロキシ呼び出し実装
- 検索結果のドメインモデル形状への変換・統合
- ページネーションパラメータのサポート

---

## 受け入れ条件

- キーワードを指定してYouTubeとTwitchの動画を横断検索できる
- キーワードを指定してYouTubeとTwitchのチャンネルを横断検索できる
- 検索結果がドメインモデル形状で返される
- サービス指定（YouTube / Twitch / 両方）のフィルタリングができる
- ページネーションが機能する
- 外部APIがエラーを返した場合、適切なエラーレスポンスが返される

---

## 次のアクション

`/develop` を実行して実装開始
