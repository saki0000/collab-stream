# User Story: 動画詳細 & チャンネル動画APIエンドポイント

> **Epic**: APIプロキシサーバー移行
> **作成日**: 2026-02-11

---

## User Story

ユーザーとして、アプリから動画詳細やチャンネルの動画一覧を取得したい。なぜなら、APIキーを安全にサーバー側で管理しつつ、必要な動画情報を取得できるようにしたいから。

---

## ゴール

YouTube API / Twitch APIへのプロキシとして動画詳細取得とチャンネル動画一覧取得のエンドポイントを実装する

---

## 依存

- US-1: 共通基盤 & サーバー骨格構築

---

## スコープ

- 動画詳細取得エンドポイント（`GET /api/videos/{id}`）
- チャンネル動画一覧取得エンドポイント（`GET /api/channels/{id}/videos`）
- YouTube APIへのプロキシ呼び出し実装
- Twitch APIへのプロキシ呼び出し実装
- 外部APIレスポンスからドメインモデル形状への変換
- エラーハンドリング（外部API障害時の適切なレスポンス）

---

## 受け入れ条件

- YouTube動画IDを指定して動画詳細をドメインモデル形状で取得できる
- Twitch動画IDを指定して動画詳細をドメインモデル形状で取得できる
- YouTubeチャンネルIDを指定してチャンネルの動画一覧を取得できる
- TwitchチャンネルIDを指定してチャンネルの動画一覧を取得できる
- 外部APIがエラーを返した場合、適切なエラーレスポンスが返される
- レスポンスがshared内のドメインモデルと一致する形状である

---

## 次のアクション

`/develop` を実行して実装開始
